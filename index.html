What is it?

Wysper is a messaging application that allows you to store your conversations on your own server.

How does it work?

Wysper is installed as a Client application and a Server application separately.
The server administrator for your chat launches the wysper-server application and gives the ip address and port to anyone they want to be able to join the chat.
Clients can launch the wysper-client application and connect using the server administrators ip address and port.

Installation

To use Wysper, follow the download links below for your platform-specific installion process, or clone our release repository at https://github.com/wysper-msg/downloads

Server

Prerequisites:
  • Ubuntu 18.04 LTS
    Note: Server is untested on other Unix systems. Feel free to try it out though!
  • Java JRE 8 or newer, OR
  • OpenJDK 1.8
To run the server, download the zip, extract where you please, and enter configuration options you want in the run.sh script. See the server README in that directory for more information.

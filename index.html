What is it?

Wysper-msg is a messaging application that allows you to store your conversations on your own server.

How does it work?

Wysper is installed as a Client application and a Server application separately.
The server administrator for your chat launches the wysper-server application and gives the ip address and port to anyone they want to be able to join the chat.
Clients can launch the client application and connect using the server administrators ip address and port.

Installation

To use Wysper, first clone our release repository at https://github.com/wysper-msg/downloads

Server

To run the server, find the run.sh file in the Server folder of the cloned repository
Edit the file and enter your port and any other configuration options you may want.

Finally, run ./run.sh to run the server.

Check out The server README for more information!

Client

To run the client, enter the Client/wysper-client folder and run java -jar client.jar to run the client.

Once the client is started, you can enter the server's ip address and port to connect and start sending messages!
